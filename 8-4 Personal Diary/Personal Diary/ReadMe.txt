pdadd
add an entity given in the file "pdadd_test.txt" 
to the diary for the date in the file "diary.txt"

pdlist [ ]
lists all entities in the diary ordered by date in the file "pdlist_test.txt"
If the start and the end date are provided in the file "pdlist_test.txt", 
it lists entities between the start and the end only.

pdshow
prints the content of the entity 
specified by the date given in  the file "pdshow_test.txt"
to the file "pdshow_test.txt"

pdremove 
removes one entity of the date given in the file "pdremove_test.txt" 
from the file "diary.txt".
